# Spring Boot Start

Template for Spring Boot Projects

## Tech Stack

- JDK: 17
- Maven: 2.x
- Spring: 2.x
  - Spring Boot
  - Spring Web
  - Spring Validation
  - Spring Security with JWT
  - Spring Data JPA
- MySQL: 8.4
- Lombok: 1.x

## Run the Service

```shell
mvn clean spring-boot:run
```

## Swagger

- http://localhost:8080/v2/api-docs
- http://localhost:8080/swagger-ui/
